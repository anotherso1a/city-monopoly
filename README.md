# 都市行走 (City Monopoly)

一款基于真实地理位置的轻量级城市大富翁游戏。

## 核心功能

- **真实定位生成地图** — 用当前位置周边的真实 POI(咖啡店、地铁站、公园)生成游戏棋盘
- **步行驱动机会卡** — 走得越远,触发越多(微信运动集成规划中)
- **完整大富翁闭环** — 买地、收租、结算、历史回放
- **地图文件分享** — 支持把专属地图分享给好友一起玩
- **海报分享** — 一键生成战绩海报保存到相册

## 技术栈

- 微信小程序原生框架
- [高德地图](https://lbs.amap.com/) — POI 搜索 / 静态图
- [MiniMax](https://platform.minimaxi.com/) — AI 生成地图内容
- 阿里云 OSS — 静态资源

## 快速开始

1. 克隆仓库,微信开发者工具导入本项目
2. 复制 `config.example.js` 为 `config.local.js`,填入高德 Key 和 MiniMax Key
3. 编译运行

详细平台配置 / 域名白名单 / 隐私声明见 [PLATFORM-SETUP.md](./PLATFORM-SETUP.md)。
